Python project for generated client code as a distributable library.
Input openapi specifications in folder openapi_specs.
Use generator [openapi-python-client](https://pypi.org/project/openapi-python-client/).
Use customization features of the generator to add logging.
Support Python 3.13 and newer.
Support async and sync calls to api.
Project to use GitLab.
Generated code not to be committed to repository.
Use pyproject.toml file to define project and dependencies.
No Makefile.
